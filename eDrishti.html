eDrishti

eDRISHTI is an Automated Public Bus Identification System for Visually Impaired. The goal of this system is to minimize the difficulties faced by the visually impaired as well as senior citizens with low eyesight for using public transportation buses on a daily basis. It facilitates them with an independent, reliable and safe transportation. The system consists of three modules: bus, server, and user module. The smart phone used by the bus conductor should be Global Positioning System (GPS) enabled which will be used by the server for finding the position of the bus. When the bus approaches a bus-stop, the user's application automatically connects to the server and gets the list of nearby available buses. The user then gets applicable bus information in the form of an audio message, using a smart phone's speaker. At the same time, the server notifies the conductor of the presence of a user on the next stop. The outcome of the system is to make this category of users independent by sending the bus information in the form of audio message. This system is intended to provide some degree of mobility to the visually impaired community.
